Principais formataçoes

Negrito ou Destaque

Para fazer um texto em negrito sem semântica usa-se a tag 'b'

E para botar em negrito de um modo que tenha semântica se usa a tag 'strong'

Itálico ou Ênfase

Para botar em Itálico sem semântica deve-se usar a tag 'i'

Para botar em Itálico com semântica deve usar a tag 'em'

Texto Marcado

Podemos criar um texto marcado usando a tag 'MARK'

O padrão do mark é amarelo, mas usando o style dentro do mark da para mudar sua cor, caso queira mudar a cor de todos os marks, colocar fora do mark

Texto Grande e Teste Pequeno

para deixar o texto grande usasse CSS e muda a forma dele direto ja que a tag big, se tornou obsoleta

Mas para fazer um texto menor da para se usar a tag small

Texto Inserido e Texto Deletado

Texto Inserido seria um texto com ênfase nele e indicar que foi adicionado depois (sublinhado)

Texto Deletado seria um texto em que se deve ler mas não considerá-lo(riscado)

Texto subscrito e sobrescrito

Texto subscrito seria para coisas tipo : X2 + 3

texto sobrescrito seria para coisas tipo H2O
