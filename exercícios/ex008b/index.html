O comando num = int(input('Qual seu nome?')) é um código de linguagem Python e para deixa-lo monoespaçado com o mostrado se usa a tag code

        
num = str(input('qual seu nome?')).strip.upper
if num == 'felipe':
    print('Uau, que nome legal')
else:
    print('Que nome bem chato hein')

Citação simples

Para fazer uma citação simples usa-se a tag 'q', ela deixa entre aspas mas com semantica

ex: Como diria Teteusu: Quem nega uma carinha amarga perde um cuzinho doce

Citação Completa

Uma citação completa é aquela em que se cita o site, livro, lugar onde se pegou tal referência

ex:

Segundo Jef Noble em seu livro: HTML para leigos

Que html é parara e piriri e tudo isso, dale
