Aprendendo Html

HTML básico

O que uma Id faz:

Id é um jeito de especificar uma parte, um H1 por exemplo no html, para depois formatar apenas ele no CSS, por exemplo:
<h1 id=teste>Criando sites em html e css</h1>
#teste{ }
Após isso escolher apenas este H1 no css, onde se chama #, ou seja 'id = #'

HTML intermediário

O que uma class faz:

Mesma função do Id porém com o class=" " se pode selecionar mais de um elemento por vez, em HTML é 'class' já em CSS é '.'
