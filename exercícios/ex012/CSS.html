CSS e seus tipos

  • CSS Inline

    • É aquela pontual, fazendo o código na linha em que se deseja a mudança. OBS: a menos recomendada pois polui a área de códigos

    Ir para a página de exemplo da CSS Inline

  • CSS Interna

    • É aquela onde se cria todo style na parte do Head e se muda tudo de uma vez na página, exemplo: muda todos h1 de uma vez, diferente da inline que muda um por vez.
    OBS: Não recomendada para quando o site houver mais de 1 página

    Ir para página de exemplo da CSS Interna

  • CSS Externa

    • É a forma de mudar o estilo em mais de uma página por vez, criando um link=css e assim em uma página criar código apenas de css e bota-lás nas páginas usando o link, assim mudando mais de uma por vez

    Ir para página de exemplo da CSS Externa
