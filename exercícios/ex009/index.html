Trabalhando com Listas

Listas Ordenadas

Usa a tag 'ol' para criar listas e a tag 'li' para adicionar o item nela, ja bota na ordem certa, sem precisar botar 1., 2., 3., etc... ex:

  1. Acordar
  2. Arrumar a cama

E tem também como personalizar estas listas usando a tag type logo após a 'ol', EX:

  1. Guardar grana
  2. Investir
  3. Saber no que investir
  4. Não ser burro

Listas não ordenadas

São listas que não precisam de uma ordem especifíca

  • Arroz
  • Feijão
  • Azeite
